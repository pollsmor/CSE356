Install Node.js: 
- https://www.digitalocean.com/community/tutorials/how-to-install-node-js-on-ubuntu-20-04

Install MongoDB:
- https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-ubuntu/

Setup NGINX:
$ apt install nginx
$ cp nginx.conf /etc/nginx/nginx.conf
$ systemctl restart nginx

Setup Postfix:
- https://www.digitalocean.com/community/tutorials/how-to-install-and-configure-postfix-on-ubuntu-20-04
$ apt install mailutils

Setup Elasticsearch:
- https://www.elastic.co/guide/en/elasticsearch/reference/8.1/deb.html
  - Also setup Kibana for visualization, but not needed.
- cp elastic.options /etc/elasticsearch/jvm.options.d/elastic.options
- Edit file /etc/elasticsearch/elasticsearch.yml to disable xpack.security.enrollment
$ systemctl restart elasticsearch
