Install Node.js: 
- https://www.digitalocean.com/community/tutorials/how-to-install-node-js-on-ubuntu-20-04

$ apt install nginx

$ npm install -g nodemon
$ npm install express
$ npm install sharedb
$ npm install ws
$ npm install @teamwork/websocket-json-stream
$ npm install rich-text
$ npm install quill-delta-to-html

Setup NGINX:
$ cp server.conf /etc/nginx/nginx.conf
$ systemctl restart nginx
